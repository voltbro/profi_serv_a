# profi_serv_a

### Installation
```
cd ~/catkin_ws/src/
git clone https://github.com/NikolayIvanovWS/profi_serv_a.git
```
### Compilation
```
cd ~/catkin_ws/
catkin_make --pkg profi_serv_a
```
### Usage

Just run configure.launch
```
roslaunch profi_serv_a start_configure_a.launch
```

### Required libraries

```
string
time
os
tqdm
os
random
```
